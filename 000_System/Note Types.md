# Note Types

## 1. Atomic Notes (200_Atomic/)
**Purpose**: Single, permanent concepts
**Naming**: Descriptive title (e.g., "Flow State Requires Clear Goals")
**Properties**: type, status, created, reviewed
**Template**: atomic-note
**Examples**: [[Deliberate Practice]], [[Stoic Dichotomy of Control]]

## 2. Structure Notes (300_Structure/)
**Purpose**: Connect atomic notes, create maps
**Types**: MOC, Index, Map, Dashboard
**Naming**: "MOC - [Topic]" or "[Topic] Index"
**Properties**: type, topics, updated
**Template**: moc-template

## 3. Literature Notes (600_Resources/Books/)
**Purpose**: Process books/articles
**Naming**: "Title - Author"
**Properties**: type, author, finished, rating
**Template**: book-note

## 4. Project Notes (400_Projects/)
**Purpose**: Active work
**Naming**: "[Project Name]"
**Properties**: type, status, deadline, next-action
**Template**: project-note

## 5. Daily Notes (500_Areas/Daily/)
**Purpose**: Capture, reflect, plan
**Naming**: "YYYY-MM-DD"
**Properties**: mood, energy, focus
**Template**: daily-note

## 6. Question Notes (200_Atomic/Questions/)
**Purpose**: Open questions driving research
**Naming**: "Q: [Question]"
**Properties**: type, status, related
**Template**: question-note

## 7. Fleeting Notes (100_Inbox/)
**Purpose**: Quick captures
**Naming**: "[Topic] - [Date]"
**Properties**: captured, source
**Template**: fleeting-note
**Lifespan**: Process within 48 hours

## 8. Area Notes (500_Areas/)
**Purpose**: Ongoing responsibilities
**Naming**: "[Area Name]"
**Properties**: type, reviewed
**Template**: area-note
