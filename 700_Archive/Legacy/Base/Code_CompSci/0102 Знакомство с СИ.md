# Базовый синтакс

Функции

```c
printf("hello,world\n");
```

Циклы 

```c
while (true)
{
    printf("hello,world\n");
}
```

кодом выше я могу выводить код снова и снова пока функция верна.

```c
for (int i = o; i < 10; i++)
{
    printf("hello,world\n");
}
```

а этим кодом мы можем выводить код N-ый раз. столько,сколько захотим.

Переменная

```c
int counter = 0;
while (true)
{
    printf("%i\n", counter);
    counter++;
}
```

а это счетчик

Условия

```c
if (x < y)
{
    printf("x is less than y\n");
}
else if (x > y)
{
    printf("x is greater than y\n");
}
else 
{
    printf("x is equal to y\n");
}
```

## Объяснение синтакса на коде

```c
#include <stdio.h>

int main(void)
{
    printf("hello,world!\n");
}
```

1. фигурные скобки "{ }" обхватывает кусок кода для того,чтобы связвать  ее с остальной программой.

2. "main" Это тоже самое что и зеленый флаг в Скрэтч. Запуск крч.

3. "printf" оператор, функция принимающая данные в ковычках. Тобишь, функция которая говорит привет.

4. " #include <stdio.h>" Дает доступ к библиотеке функций.

> "make (название файла" потом "./(название файла)"  Для компиляции кода. 

5. "int" - integer. четкая цифра. Тип переменной 

6. " ; " завершение строки.

Код №2

```c
#include <cs50.h>
#include <stdio.h>


int main (void)
{
    string s = "Hannah";
    printf ("hello,%s\n,s");
}
```

1. "%" -  это заполнитель. тем самым комп поймет что нужно вывести строку(string) s.

2. "string" - строка. но ее не существует на Си. Поэтому мы добавили костыль " cs50"

```c
#include <cs50.h>
#include <stdio.h>


int main (void)
{
    string s = GetString()
    printf ("hello,%s\n");
}
```

1. "GetString" - это запрос данных,в нашем случае это строка,а если еще точнее то имя. Тем самым мы сделали наш код более универсальным.

> "&&" - И. 7 и 9 = 7  && 9
> 
> " || " - or/или.  7 или 9 = 7 || 9
> 
> "\n" - символ новой строки. перевод курсора на след строку.
