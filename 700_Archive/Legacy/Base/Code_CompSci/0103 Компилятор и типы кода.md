# Что?Компилятор?
Компилятор - берет исходный код и образует его в объектный.
## Исходный код
Исходный код - код,написанный человеком на языке программирования(Python,C,Ruby,Java etc)
### Объектный код
Объектный код - мост между исходным кодом и машинным. Во многом это просто каша символов.
#### Машинный код 
Машинный код - 0101101001010101010. это уже готовая программа,ждет своего часа.
