# Наполнители 
1. %с - для вывода символа.
2. %i - для вывода числа.
3. %f - float and double.
4. %s - string
5. %lld - long long
## Управляющие последовательностью 
1. \n - новая строка кода
2. \r
3. \t - табуляция
4. \'
5. \"
6. \\
7. \0
