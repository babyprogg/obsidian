# Базовое знание 
1. Массив - последовательность символов. 
2. Когда массив обьявляется то он выглядит так:
type name [size];
## Аргументы командной строки 
int main(int argc,string argv[])
Здесь argc это сколько слов,символов и тд в линии, а argv[]
это сколько памяти мы бронируем.
