1. malloc - запрос памяти у компа
2. sizeoff - сколько нужно в байтах 
3. sizeoff(int) - столько,сколько нужно инту 
4. int* x - адрес инта икса
5. *x = 42 иди по этому адресу икса и помести туда 42.
6. &x - иди и разузнай адрес икс. противоположное звездочке.
7. realloc - он автоматически настраивает память запрашиваемого данного. если короткая то короткая,еси длинная то длинная.
