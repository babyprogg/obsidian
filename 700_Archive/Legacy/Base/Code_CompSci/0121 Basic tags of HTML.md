# Базовое понятие

Тэги хтмл это тоже самое что и функции си,питона и тд. они идут вместе с <  > and </  >

## Базовые тэги

1. <meta> 
    1. Она нужна для того,чтобы сказать браузеру какой тип символов мы используем. 
	    Она обычно выглядит так:<meta charset="UTF-8">
    2. <meta name="description" content="Крутой сайт ау мау">
	    Эта мета нужна чтобы дать браузеру описание нашего сайта которые браузеры могут использовать.
	В этих тегах есть атрибуты как "name,charset,content" и тд. Это все своя информация которую мы можем внедрить в сайт.
2. <h1> 
	1. Это нужно чтобы дать заголовок сайту.
	2. Можно дать от одного до шести. Отличаются они размером.
3. <p> 
	1. Нужен для создания параграфа. 
	2. Они автоматически отделяеются от других параграфов. Не нужно как в заголовках делить по цифрам. 
4. <b> 
	1. Нужен для того чтобы сделать текст жирнее.
	2. Для этого нужно перед словом создать тэг. Примерно так: <b>Maxat</b>
5. <i>
	1. Нужен чтобы текст был курсивным.
	2. Для этого нужно так же как и болд тэгом но уже другой тэг.
*Можно совмещать эти два тэга. Так: <b><i>Maxat</i></b>. Тогда текст будет и жирным и курсивным.*
*В каком порядке располежены в коде,то такие же порядки будут и в сайте.*
*ХТМЛ автоматом убирает белые линии в коде при показе на сайте.*
6. </br>
	1. Нужно для того чтобы создать пространство между параграфами,заголовками и тд. 
7. <hr/>
	1. Нужно для создания горизонтальной линии между строками.
8. <big>
	1. Нужно чтобы сделать слова больше.
9. <small>
	1. Нужно для того чтобы сделать слова меньше.
10. <sub>
	1. Нужен чтобы сделать цифры как основу логарифма. Чуть ниже.
11. <sup>
	1. Нужен чтобы сделать цифру как степень.
