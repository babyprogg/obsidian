# Каркас сайта
Любой сайт состоит из:
1. Заголовок.
	Заголовок может быть навигатором по сайту,введением и тд.
2. Основа
	Это мясо и кости сайта. Здесь могут параграфы, статьи, фотки и тд.
3. Конец
	Здесь могут быть ссылки на соц сети,библиография и тд.
## Тэги
1. <nav>
	Нужен для размещения всяких ссылок и тд.
2. <article>
	Нужен для размещения статьи.
3. <section>
	Для создания секций статьи.
4. <aside>
	То,что будет показываться пользователям но не относится к кор инфе сайта.
