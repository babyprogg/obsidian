## Как делаются видео
<video src="адрес видео" loop autoplay poster="адрес картинки"controls width="300">текст который будет на дисплее есть не получится открыть видео.</video>
controls - дает контроль над видео. переключать скорости, перематывать и тд.
poster - превью видоса.
autoplay - при открытии сайта видос автоматом будет проигрываться.
loop - повторяшка видоса.
