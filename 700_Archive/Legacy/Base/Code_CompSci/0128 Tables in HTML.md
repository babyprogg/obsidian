## Тэги
<table> - тег таблицы
<tr> - линия таблицы
<td> - инфа внутри таблицы
<caption> - название таблицы
<th> - заголовок линии таблицы
<thead> - заголовоки,название ау мау, голова крч
<tbody> - то,где хранится весь контент таблицы.
<colspan> - сколько коллон будет занимать
