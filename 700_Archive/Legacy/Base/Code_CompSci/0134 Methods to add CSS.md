## Internal
Через тэг <style>
	<style>
	h1{color:green;
	}
	</style>
### External
Связать файл с external.css file через тэг <link>
	h1{color:green;
	}
	<link rel="stylesheet" type="text/css"href="название файла где находится css файл/название css файла.css">
