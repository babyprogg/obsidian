# Атрибуты
1. text-decoration:none. Избавляет нас от линии под ссылкой.
2. color:black. 
3. a:hover{
color:red;
} Какой цвету будет отображатся когда будешь наводить курсором на ссылку.
4. a:active{
color:green;
} Цвет когда кликнешь на ссылку
5. a:visited{
color:purple;
} Цвет ссылки когда вы уже посетили ее.
