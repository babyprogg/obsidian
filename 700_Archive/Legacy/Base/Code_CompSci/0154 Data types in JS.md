## Применение 
1. Old way
	const name = 'Max'
	const age = '24'
	console.log('My name is '+ name +' and I am '+age);
	Чтобы использовать данные внутри переменной нужно брать их в '+ variable +'
2. New way
	const name = 'Max'
	const age = '24'
	console.log('My name is ${name} and I am ${age}');
	Чтобы использовать данные внутри переменной нужно брать их в ${variable}
