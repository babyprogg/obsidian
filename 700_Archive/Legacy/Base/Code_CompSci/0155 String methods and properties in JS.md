# Methods
*Все метдоы делаются через console.log*
1.  toUppercase - выводит переменную с капсом
2. toLowercase - все маленькие
3. length - длинна символов
4. substring(0,5) - выводит переменную начиная с 0 символа и окончивается на 5.
5. split('че то или ничего. смотря че хотим') - делит переменную по данным в сплите
