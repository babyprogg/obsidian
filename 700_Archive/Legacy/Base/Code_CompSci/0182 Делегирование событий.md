# Зачем и идея

Идея в том,чтобы вместо обработки кучи похожих на друг друга событий,мы ставим обработчика на их общего предка. Это экономит время и делает процесс работы легче.

## Применение на примерах

![[Pasted image 20230320170843.png]]
Наша цель - помечать красным каждый квадрат, который кликнули.
Вместо того чтобы писать код **onclick** на каждый квадрат, мы вешаем обработчик на их отца,на целевой элемент.([[0181 Event bubbling and Capturing in JS#event.target]])
Для этого мы будем использовать **event.target** ([[0181 Event bubbling and Capturing in JS#event.target]])
Код будет таким:

```javascript
table.onclick = function(event) {
  let target = event.target; // где был клик?

  if (target.tagName != 'TD') return; // не на TD? тогда не интересует

  highlight(target); // подсветить TD
};
```
