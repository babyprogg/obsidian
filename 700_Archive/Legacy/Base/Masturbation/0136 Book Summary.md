# Read
Мортимер адлер 
## Заметки
Цитаты,интересные идеи,мысли автора,ресурсы которые автор советует/пользуется,самые главные мысли главы.
### Саммари
1. Нон фикшн
	1. Главная идея
	2. 3-5 Основные идеи,моменты
	3. Примеры и тд 
2. Фикшн
	1. Пять персонажей(гг) которые(мотивация персов и двигатель сюжета)но(что нужно преоделеть чтобы добиться желаемого) так(что они сделали для решения) потом(конец истории)
NON-FICTION VS FICTION : NF : Big idea vs F: main character | NF : Main points F : Major events | NF : Takeaway F : Conclusion
#### Пряности 
Истории или примеры автора которые повлияли на тебя,сподвигнули м размышлению и тд.
