Тем самым сделать наш заголовок сразу с ссылкой.
**Ссылки откроются на вашей же странице. Если хотите чтобы ссылка открылась в новой вкладке то нужно сделать так.
<a href="ссылка на сайт" target="_blank
"> Текст к ссылке</a>

Для того чтобы дать ссылку на вторую страницу вашего сайта то нужно сделать так
<a href="ссылка на страницу.html"> Текст к ссылке</a>
Но это если страницы находятся в одной директории. 
Если в разных то нужно сделать так
<a href="название директория/ссылка на страницу.html"> Текст к ссылке</a>
Если нужно дать ссылку на файл то делаем так
<a href="ссылка на файл.формат файла"> Текст к ссылке</a>
