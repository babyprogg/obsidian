4. <input type="date"/>
5. <input type="range"/>
6. <input type="file"/>  
7. <input type="checkbox"/>

## Факью
value - текст который будет отображатся в форме.
textarea - окно для текста
Если у radio одинаковые имена то можно выбрать лишь одну.
forms - контейнер для интпутов
