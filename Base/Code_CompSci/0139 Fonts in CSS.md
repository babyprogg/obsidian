# Fonts
1.  Web fonts. Есть во всех браузерах,не нужно парится
2. Own fonts. Нужно линкить через стайлшитс. 
## Types of fonts 
1.  Font families. Комбинация шрифтов(гугли)
### Атрибуты(гугли)
1. font-size:16px;
2. font-weight:bold:(жирность шрифта крч)
3. font-family: Arial,Helvetica,Sans serif
4. font-style:italic
5. text-decoration:underline(че то с шрифтом,типо линия и тд)
6. text-transfom:uppercase
7. letter-spacing:0.2em(расстояние между буквами)
8. word-spacing:1em(расстояние между словами)
