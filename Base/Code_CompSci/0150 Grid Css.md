#  Syntax
1. grid-column-start: 1(-1). Окуда колонны начнутся.
2. grid-column-end: 1(-1). Где они закончатся.
3. span. Берет все и до конца.
4. grid-column:1 / 4. Два в одном.
5. grid-row: 3. По вертикали.
6. grid-area: 1 / 1 / 2 / 5. Все в одном(row start/column start/row end/column end)
7. order: откуда начинается для отдельного элемента.
8. grid-template-columns: ширина плюс высота расположение где будут эдементы.
9. grid-template-columns: repeat(5,20%)
10. fr - дробные части
