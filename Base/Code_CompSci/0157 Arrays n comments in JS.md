## Arrays(массивы)
*Это переменные которые хранят в себе несколько перменных.*
*Способы создания массива*
	1. Конструктор(constructor) - new
	const numbers = new Array(1,2,3,4,5);
	2. const fruites = ['apples','oranges','bananas'];
	Более используемый способ. 
**В Js можно хранить разные типы данных в один массив.**
*Работа с массивами*
1. Если мы хотим использовать только что то одно из массива томы делаем так:
	const fruites = ['apples','oranges','bananas'];
	console.log(fruites[1]);
	Так мы будем работать чисто с oranges 
**Массивы начинаются с нуля везде**
2. Можно добавлять в массив данные на ходу 
	const fruites = ['apples','oranges','bananas'];
	fruites[3] = ['grapes'] 
	Но нельзя использовать один массив два раза. Типо так:
	fruites = ['pears'] Это уже ошибка.
3. Так же,добавлять данные в массив таким способом не очень эффективно. Есть такой вариант:
	- fruites.push('mangos'); Добавить данные в конец массива.
	- fruites.unshift('strawwberries'); Добавить данные в начало массива.
	- fruites.pop(); Удаляет последнее из массива.
4. Если ты хочешь чекнуть существует ли какой либо массив то:
	console.log(Array.isArray(fruites));
5. Если ты хочешь узнать порядок данного в массиве то:
	console.log(fruites.indexOf('oranges'));
