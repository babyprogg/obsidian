- Выражения JSX должны иметь один родительский элемент, потому что, при конвертации JSX в код JavaScript, он должен быть обернут вокруг одного элемента. Несколько элементов не могут существовать без общего родителя.
- Можно писать джиэс прямо в {} типо {1 + 1}, {name}, {x ? 'Yes' : 'No'}
- Пропсы это крч переменные значения которые можно менять моментально в коде. типо 
``` javascript
const Header = (props) => {

  return (

    <header>

      <h1>{props.title}</h1>

    </header>

  )
}
```
``` javascript
    <div className="container">

      <Header title='Bruh' />  

    </div>
```
можно еще делать дефолт пропсы типо 
``` javascript
Header.defaultProps = {

    title: 'Bruh'

}
```

- пропс тайпы это крч типы данных которые должны быть пропсы. делается это так. это все пример, можно менять почти все.
```js
import PropTypes from 'prop-types'

Header.PropTypes = {

    title: PropTypes.string

}
```  

стайлинг производится в {}, все штучки с дефизом пишутся по верблюдский. background-color => backgroundColor
