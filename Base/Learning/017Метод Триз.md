# Метод Триз
1. Мозговой штурм - обсуждения решение проблемы, даже самые странные решения.
2. Синектика/Метод Аналогий - сравнения и нахождения сходства между объектами.
3.  Морфологический Анализ - поиск решения для отдельных частей задачей с целью последующего сочетания этих решённый.(проблема и варианты решения проблемы и его аспекты) 
4. Метод Фокальных Объектов - применение свойств объектов к другому объекту/-ам. (Смотреть под другим углом)
5. Метод Робинзона - Разнообразное применение любых объектов. (Диплом для стола чтоб не шатался)
6. Системный Оператор - связывать объектив в систему (поиск связей с объектами которые могут стать одной системой. яблоко+дерево+листья=яблочная дерево)
